# 更新日志 (Changelog)

## [v1.2.0] - 最新更新

### ✨ 新特性 (Features)
- **高级后台管理中心**：重构了管理后台，引入“草稿-发布”机制。所有的修改（内容、排版、配色）现在可以作为草稿预览，点击“保存发布”后才会同步到前台。
- **自定义板块管理**：
  - 在排版设置中支持**新增自定义板块**。
  - 支持对板块进行**重命名**（限制 8 个字符）。
  - 支持拖拽排序、显隐切换以及**删除板块**。
- **全局配色与预设主题**：
  - 开放了网页背景色、全局文字色、强调色和卡片背景色的完全自定义权限。
  - 新增多种**高级预设配色**：Apple 暗黑、Apple 极简白、Vercel 黑客、Stripe 商业、Spotify 沉浸、Linear 质感、Notion 极简、Discord 活泼等。
  - **实时主题预览**：后台切换配色时，管理界面自身也会实时变色，实现所见即所得。
- **一键导入配置 (JSON)**：各个后台配置模块底部新增了文件导入功能，支持直接上传 `.json` 文件快速覆盖当前草稿数据，实现“白痴化配置”。
- **作品详情沉浸式弹窗**：前台作品卡片点击“查看详情”后，不再进行页面跳转，而是呼出带有毛玻璃遮罩（`backdrop-blur`）的高颜值大尺寸弹窗，配合 Framer Motion 实现丝滑过渡。

### 🔐 安全与权限 (Security)
- **管理员认证机制**：为 `/admin` 后台入口增加了登录拦截。仅当输入正确的账号（`user`）与密码（`112233`）时才允许进入，并提供安全的登出功能。
- 移除了前台主页导航栏的“设置”快捷入口，保证前台作为纯粹的展示页面。

### 🐛 修复与优化 (Fixes & Optimizations)
- **修复首屏排版重叠**：优化了 Hero 组件的底部边距与 Flex 布局，彻底解决了“向下滚动”指示器与个人简介文字重叠的问题。
- **修复浅色模式文字不可见**：移除了前台组件（Hero, About, Projects, Footer 等）中硬编码的 Tailwind 白色文本类（如 `text-white/60`），重构为透明度继承（如 `opacity-60`），完美兼容各种浅色与深色主题。
- **修复动画库冲突导致白屏**：清理了 `Projects.tsx` 中重复导入 `framer-motion` 导致的致命语法错误，恢复了前台的正常渲染。

## [v1.1.0]

### ✨ 新特性 (Features)
- **引入数据持久化**：引入 `zustand` 结合 `localStorage`，实现前台所有展示数据（基本信息、项目列表）的动态化与本地持久保存。
- **基础管理后台**：建立了 `/admin` 路由，允许用户通过界面表单直接修改姓名、简介、技能与项目卡片信息。

## [v1.0.0] - 初始版本

### ✨ 新特性 (Features)
- **苹果风极简框架**：搭建了基于 React + Tailwind CSS 的现代个人网站框架。
- **核心组件**：完成 Navigation、Hero、About、Projects 和 Footer 五大模块的响应式设计。
- **视觉与动画**：引入深色模式、毛玻璃导航栏，并集成 `framer-motion` 实现首屏淡入与滚动视差等丝滑交互动画。
